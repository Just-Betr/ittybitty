# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-02-03
### Added
- Display the app version in the top bar using the crate version.
- Changelog file for tracking releases.

### Changed
- Enforce file selection using `only_files` at add time without post-add updates.
- Widened DOWN and UP columns in the torrent list for better speed visibility.

### Removed
- Removed the post-add initialization wait and `update_only_files` step.
